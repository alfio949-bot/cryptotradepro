📖 Manuale Utente — CryptoTrade Pro
🚀 Quick Start
Registrati → accedi → seleziona una coppia nella Watchlist → analizza il grafico → inserisci un ordine dal pannello Order Form.
📊 1. Dashboard Trading
• Grafico: aggiornato in tempo reale via WebSocket Binance (intervallo configurabile: 1m, 5m, 15m, 1h, 4h, 1D).
• Order Book: visualizza i migliori 12 livelli di bid/ask in tempo reale.
• Order Form: supporta ordini Market, Limit e Stop-Limit. Seleziona la leva (1x–100x), inserisci la quantità e conferma.
• Posizioni: tab inferiore con PnL live calcolato tick-by-tick. Il pulsante "Chiudi" esegue la chiusura al prezzo corrente.
• Ordini Pendenti: gestisci e cancella gli ordini Limit non ancora eseguiti.
• Storico: registro completo con volume acquisti/vendite, commissioni e P&L netto.
👁 2. Watchlist
• Filtra per categoria (DeFi, Layer1, Layer2, Stablecoin, Meme, Top 20) o usa la ricerca testuale.
• Clicca ★ per aggiungere/rimuovere dai Preferiti.
• I prezzi si aggiornano ogni 10 secondi tramite Binance 24hr ticker batch API.
• Clicca su un asset per aprirlo direttamente nel pannello Trading.
🌐 3. DeFi Hub
• Wallet: aggiungi indirizzi EVM/Solana per monitorare il saldo on-chain.
• Token: visualizza il portafoglio token per ogni wallet connesso.
• Grafico DeFi: powered by GeckoTerminal — analisi pool AMM in tempo reale.
• Notifiche: configura alert prezzo e movimento whale.
📈 4. Mercati Azionari (TradFi)
• Paper Trading con €100.000 di capitale virtuale iniziale.
• Spread markup configurabile (0.1%–0.5%) gestito dal pannello Admin.
• Acquista e vendi azioni simulate con grafici TradingView integrati.
• Storico ordini completo con calcolo automatico del P&L per posizione.
🔐 5. Sicurezza
• 2FA: attiva l'autenticazione a due fattori via TOTP (Google Authenticator).
• API Keys: le chiavi vengono cifrate localmente con AES-256 prima del salvataggio.
• Sessioni: visualizza e termina le sessioni attive da altri dispositivi.
• PIN: imposta un PIN di accesso rapido per sessioni brevi.
⚙️ 6. Impostazioni
• Lingua, valuta di riferimento, slippage e tema (light/dark).
• Notifiche push, sound alert e alert prezzo personalizzati.
• Connessione exchange tramite API key (Binance, Bybit, OKX, ecc.).
• Formspree endpoint per ricevere i feedback beta via email.
🐛 7. Beta Feedback
• Clicca "🐛 Beta Feedback" nella barra di navigazione per inviare segnalazioni.
• Classifica il tipo (Bug, Feature Request, Miglioramento, Performance) e la severità.
• Il feedback viene salvato localmente e (se configurato) inviato via email all'admin.
🔒 Privacy: tutti i dati (wallet, API keys, preferenze) sono archiviati esclusivamente nel tuo browser (localStorage). Nessun dato viene trasmesso a server terzi senza esplicita autorizzazione.
